Proof

We will use induction on \(n\text{.}\) For \(n = 1\) the theorem is trivial. Assume that the theorem is true for all \(k\) such that \(1 \leq k \leq n\text{.}\) Then \begin{align*} z^{n+1} & = z^n z\\ & = r^n( \cos n \theta + i \sin n \theta ) r( \cos \theta + i\sin \theta )\\ & = r^{n+1} [( \cos n \theta \cos \theta - \sin n \theta \sin \theta ) + i ( \sin n \theta \cos \theta + \cos n \theta \sin \theta)]\\ & = r^{n+1} [ \cos( n \theta + \theta) + i \sin( n \theta + \theta) ]\\ & = r^{n+1} [ \cos( n +1) \theta + i \sin( n+1) \theta ]. \end{align*}
