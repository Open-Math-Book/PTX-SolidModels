Let \begin{equation*} S = \{ am + bn : m, n \in {\mathbb Z} \text{ and } am + bn \gt 0 \}. \end{equation*} Clearly, the set \(S\) is nonempty; hence, by the Well-Ordering Principle \(S\) must have a smallest member, say \(d = ar + bs\text{.}\). We claim that \(d = \gcd( a, b)\text{.}\). Write \(a = dq + r'\) where \(0 \leq r' \lt d\text{.}\). If \(r' \gt 0\text{,}\), then \begin{align*} r'& = a - dq\\ & = a - (ar + bs)q\\ & = a - arq - bsq\\ & = a( 1 - rq ) + b( -sq ), \end{align*}
