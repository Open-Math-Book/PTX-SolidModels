Subsets of \(A \times B\) are called relations. We will define a mapping or function \(f \subset A \times B\) from a set \(A\) to a set \(B\) to be the special type of relation where \((a, b) \in f\) if for every element \(a \in A\) there exists a unique element \(b \in B\text{.}\). Another way of saying this is that for every element in \(A\text{,}\), \(f\) assigns a unique element in \(B\text{.}\). We usually write \(f:A \rightarrow B\) or \(A \stackrel{f}{\rightarrow} B\text{.}\). Instead of writing down ordered pairs \((a,b) \in A \times B\text{,}\), we write \(f(a) = b\) or \(f : a \mapsto b\text{.}\). The set \(A\) is called the domain of \(f\) and \begin{equation*} f(A) = \{ f(a) : a \in A \} \subset B \end{equation*} is called the range or image of \(f\text{.}\). We can think of the elements in the function's domain as input values and the elements in the function's range as output values.
