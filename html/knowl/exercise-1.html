Exercise1

Suppose that \begin{align*} A & = \{ x : x \in \mathbb N \text{ and } x \text{ is even} \},\\ B & = \{x : x \in \mathbb N \text{ and } x \text{ is prime}\},\\ C & = \{ x : x \in \mathbb N \text{ and } x \text{ is a multiple of 5}\}. \end{align*}

Describe each of the following sets.

  1. \(A \cap B\)

  2. \(B \cap C\)

  3. \(A \cup B\)

  4. \(A \cap (B \cup C)\)

Hint

(a) \(A \cap B = \{ 2 \}\text{;}\) (b) \(B \cap C = \{ 5 \}\text{.}\)
