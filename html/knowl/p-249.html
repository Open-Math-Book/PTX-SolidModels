For all integers \(n \geq 3\text{,}\), \(2^n \gt n + 4\text{.}\). Since \begin{equation*} 8 = 2^3 \gt 3 + 4 = 7, \end{equation*} the statement is true for \(n_0 = 3\text{.}\). Assume that \(2^k \gt k + 4\) for \(k \geq 3\text{.}\). Then \(2^{k + 1} = 2 \cdot 2^{k} \gt 2(k + 4)\text{.}\). But \begin{equation*} 2(k + 4) = 2k + 8 \gt k + 5 = (k + 1) + 4 \end{equation*} since \(k\) is positive. Hence, by induction, the statement holds for all integers \(n \geq 3\text{.}\).
