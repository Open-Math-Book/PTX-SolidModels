Let \(X\) be a set. Define the power set of \(X\text{,}\), denoted \({\mathcal P}(X)\text{,}\), to be the set of all subsets of \(X\text{.}\). For example, \begin{equation*} {\mathcal P}( \{a, b\} ) = \{ \emptyset, \{a\}, \{b\}, \{a, b\} \}. \end{equation*} For every positive integer \(n\text{,}\), show that a set with exactly \(n\) elements has a power set with exactly \(2^n\) elements.
