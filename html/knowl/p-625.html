Conversely, if we are given a rectangular representation of a complex number, it is often useful to know the number's polar representation. If \(z = 3 \sqrt{2} - 3 \sqrt{2}\, i\text{,}\), then \begin{equation*} r = \sqrt{a^2 + b^2} = \sqrt{36 } = 6 \end{equation*} and \begin{equation*} \theta = \arctan \left( \frac{b}{a} \right) = \arctan( - 1) = 315^{\circ}, \end{equation*} so \(3 \sqrt{2} - 3 \sqrt{2}\, i=6 \cis 315^{\circ}\text{.}\).
