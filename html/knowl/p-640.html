The first thing to notice is that any number \(a\) can be written as the sum of distinct powers of 2; that is, we can write \begin{equation*} a = 2^{k_1} + 2^{k_2} + \cdots + 2^{k_n}, \end{equation*} where \(k_1 \lt k_2 \lt \cdots \lt k_n\text{.}\). This is just the binary representation of \(a\text{.}\). For example, the binary representation of 57 is 111001, since we can write \(57 = 2^0 + 2^3 + 2^4 + 2^5\text{.}\).
